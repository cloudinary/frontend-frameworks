lazyload

Description

Image loads once it is in a certain margin in the viewport. This includes vertical and horizontal scrolling.

Parameters
Name Type Description
rootMargin string

The root element's bounding box before the intersection test is performed defaults to 0px
threshold number

A number which indicate at what percentage of the images's visibility the image should load. The default is 0.1 which indicated 100%
Returns
Examples
plugins=[(lazyload('0px', 0.25))]
